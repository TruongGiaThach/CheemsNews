<!-- GETTING STARTED -->
## Getting Started

Install flutter version 2.2.1 or < 3.0.0 following https://flutter.dev/docs/get-started/install


### Installation

1. Clone the repo
2. Create your project on firebase and add into repo
3. Install library in pubpect.yml
4. Run project


<!-- USAGE EXAMPLES -->
## Usage

## Contributing
